Pomodoro Timer
==============

AngularJS pomodoro timer

##Installation

### Requirements

- git
- node.js 0.10+

### Installation

Install all dependecies via the following commands:

    npm install

In an other terminal start the server:

    npm start

You're ready to go! Just open your favorite browser and go to:

    http://127.0.0.1:8000/app/
